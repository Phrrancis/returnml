# ReturnML Landing

Next.js marketing site for ReturnML — real-time return-risk scoring at checkout with uplift-based actions. Stack: Next.js App Router, Tailwind, TypeScript, Vercel Analytics.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build / lint

```bash
npm run lint
npm run build
```

## Deploy

Deploy to Vercel or any Node 18+ host. Set `NODE_ENV=production` for prod builds.
